
# Build NFD

Execute the following commands to build NLSR:

<li>./waf configure</li>
<li>./waf</li>
<li>sudo ./waf install</li>

<pre>
root@bertopeng17-ThinkPad-T520:/home/bertopeng17/NFD# <b>./waf configure</b>
</pre>
<pre>
root@bertopeng17-ThinkPad-T520:/home/bertopeng17/NFD# <b>./waf</b>
</pre>
